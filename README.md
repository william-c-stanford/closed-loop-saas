# closed-loop-saas

A Claude Code plugin marketplace for building and maintaining closed-loop SaaS products.

## Install

```shell
/plugin marketplace add williamstanford/closed-loop-saas
```

## Plugins

| Plugin | Description |
| :--- | :--- |
| [context-gardening](./plugins/context-gardening) | Agent-legible repository knowledge management — CLAUDE.md navigation maps, nested per-module docs, and an AI gardener that keeps docs in sync with code. |

## Usage

Install individual plugins:

```shell
/plugin install context-gardening@closed-loop-saas
```
